# Android Stopwatch App

A simple Stopwatch App developed using Java and XML in Android Studio as part of the Oasis Infobyte Internship (OIBSIP).

## Features

- Start Stopwatch
- Stop / Pause Stopwatch
- Resume Stopwatch
- Reset Stopwatch
- Millisecond Precision
- Real-Time Timer Updates
- Hours, Minutes, Seconds, and Milliseconds Display
- Simple and User-Friendly Interface

## Technologies Used

- Java
- XML
- Android Studio
- Android SDK
