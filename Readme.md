# OIBSIP - Android App Development Internship

This repository contains the Android App Development tasks completed as part of the Oasis Infobyte Internship (OIBSIP).

## Completed Tasks

### Task 3 - Calculator

A simple Calculator Application developed using Java and XML.

#### Features

* Addition
* Subtraction
* Multiplication
* Division
* Decimal Support
* Clear Button
* Backspace Button
* Expression Display
* Division by Zero Error Handling

**Project Folder:** `AndroidApp-Task3-Calculator`

---

### Task 4 - Quiz Application

An interactive technology Quiz Application with multiple-choice questions and score tracking.

#### Features

* Welcome Screen
* 10 Technology Questions
* Four Answer Options
* Question Counter
* Random Question Order
* Correct and Incorrect Answer Feedback
* Score Tracking
* Result Screen
* Correct and Incorrect Answer Count
* Restart Quiz Button

**Project Folder:** `AndroidApp-Task4-QuizApp`

---

### Task 5 - Stopwatch

A functional Stopwatch Application developed using Java and XML.

#### Features

* Start Timer
* Pause Timer
* Reset Timer
* Time Display
* Stopwatch State Management
* Android Activity Lifecycle Handling

**Project Folder:** `AndroidApp-Task5-Stopwatch`

---
