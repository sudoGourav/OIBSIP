# Android Calculator App

A simple Calculator App developed using Java and XML in Android Studio as part of the Oasis Infobyte Internship (OIBSIP).

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal Support
- Clear Button
- Backspace Button
- Expression Display
- Error Handling (Division by Zero)

## Technologies Used

- Java
- XML
- Android Studio
- Android SDK
