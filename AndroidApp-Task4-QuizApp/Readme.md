# Android Quiz Application

A simple Quiz Application developed using Java and XML in Android Studio as part of the Oasis Infobyte Internship (OIBSIP).

## Features

* Welcome Screen
* 10 Technology Questions
* Multiple Choice Questions
* Four Answer Options
* Question Counter
* Random Question Order
* Correct Answer Feedback
* Incorrect Answer Feedback
* Score Tracking
* Result Screen
* Correct and Incorrect Answer Count
* Restart Quiz Button

## Technologies Used

* Java
* XML
* Android Studio
* Android SDK
